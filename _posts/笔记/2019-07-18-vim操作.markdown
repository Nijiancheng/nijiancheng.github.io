### 安装 vim

​ apt-get install vim

### liunx 下的 vim 操作

​ / ：搜索内容

​ i ：进入插入模式

​ esc：退出当前操作

shift + ；：输入命令

​ q + <Enter>： 推出 vim

​ q！+ <Enter>：强制退出

​ w + <Enter>：保存当前内容

​ wq + <Enter>：保存并退出

help + <Enter>: 查看帮助

选中后就可以用编辑命令对其进行编辑，如
d 删除

​ y 复制 （默认是复制到"寄存器）

​ p 粘贴 （默认从"寄存器取出内容粘贴）
