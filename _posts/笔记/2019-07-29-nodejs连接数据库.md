### nodejs连接数据库

###### 	导入mysql：

`var mysql = require('mysql');`

###### 	连接数据库：

  ```js
var connection = mysql.createConnection({
      host     : '127.0.0.1',
      user     : 'root',
      password : '123456',
      database : 'xdh'
  });
  ```

###### 开启数据库：

`connection.connect();`

###### 拼sql语句并发送：

```js
var sql = 'UPDATE user SET age="19" WHERE id="1"';
connection.query(sql, function (err, result) {
}
```
